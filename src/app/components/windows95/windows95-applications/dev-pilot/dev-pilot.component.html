About this application

DevPilot is a local development dashboard designed to simplify the management of software projects consisting of multiple parts such as frontends, backends and supporting services. Everything can be managed from a single interface.

System requirements
Component Minimum
Framework Qt 6.x
OS Windows / Linux / macOS
Status Active Development
Main features
1 Create and organize software projects
2 Add executable modules to projects
3 Launch modules directly from the dashboard
4 Open projects in terminals, IDEs and file explorers
5 Save and manage useful code snippets
