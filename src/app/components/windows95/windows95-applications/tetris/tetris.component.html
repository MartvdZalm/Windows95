About this application

A classic Tetris game built entirely in Java, written as a personal project just for fun. Guide falling tetrominoes to complete horizontal lines and rack up your high score before the stack reaches the top!

System requirements
Component Minimum
Java JDK 17 or higher
Build tool None (download the .jar)
OS Any OS with Java support
How to run
1 Install Java 17 or higher if you haven't already — download it here
2 Download tetris.jar from the latest GitHub Release
3 Double-click the JAR file, or run java -jar tetris.jar in your terminal
4 Enjoy the game!
Controls
← → Move left / right
↑ Rotate piece
↓ Soft drop
Space Hard drop
P Pause / Resume
Esc Quit game
