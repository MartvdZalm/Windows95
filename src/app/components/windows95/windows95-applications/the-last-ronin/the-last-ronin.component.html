About this application

TheLastRonin is a modern desktop music player built with Qt. It allows users to play local audio files, create and manage playlists, organize music libraries, and access YouTube Music integration from a clean and responsive interface. The project is currently under active development, with new features and improvements being added regularly.

Features
Feature Description
Audio Playback Play local music files
Playlists Create and manage custom playlists
Music Library Browse and organize your collection
YouTube Music Integrated music streaming support

How to run
1 Download the latest release from GitHub.
