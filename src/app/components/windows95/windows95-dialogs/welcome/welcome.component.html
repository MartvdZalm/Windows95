Welcome to Windows 95

Thanks for checking out my website! This is a fully functional Windows 95 experience built with Angular. Here's how to navigate:

  • Notepad - Contains my full portfolio with skills, experience, and projects (mobile-friendly)
  • Terminal - Try the shutdown sequence to access a command-line interface
  • Desktop Icons - Explore various aspects of Windows 95
Note: Most features are designed for larger screens (laptops/desktops). For mobile users, the portfolio in Notepad is fully accessible, but other interactive elements may not work as intended on small screens.
